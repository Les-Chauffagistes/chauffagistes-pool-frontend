Chauffagistes – Informations

Modes de minage, fonctionnement des gains et vision du projet. Latence optimisée pour la France grâce à notre hébergement redondé en Île-de-France.

Infrastructure

Notre architecture repose entièrement sur la stack open-source ckpool, reconnue pour sa stabilité et sa faible latence.

Pour garantir une connexion fiable en France, nous utilisons un proxy Stratum (ckproxy) hébergé chez OVH en Île-de-France. Ce proxy gère automatiquement la bascule vers l’un de nos deux serveurs ckpool physiquement séparés.

Mode actif → passif

Le proxy fonctionne en actif/passif : tant que la pool primaire répond, tout le trafic passe dessus. En cas d’incident, un failover immédiat bascule les mineurs vers la pool secondaire.

Pool primaire

Hébergée sur un cluster Proxmox haute disponibilité (HA), elle repose sur deux nœuds Bitcoin Core 28.1 : un nœud public + un nœud de backup interne. C’est notre environnement principal, pensé pour assurer un uptime maximal.

Pool de secours

Hébergée elle aussi sur Proxmox mais sans HA, avec un nœud Bitcoin Core conteneurisé. Nous considérons actuellement ce système comme un mode dégradé, utilisé uniquement si la pool primaire n’est pas disponible.

Notre objectif : améliorer continuellement notre stack pour proposer une disponibilité toujours plus élevée, une latence très faible et la meilleure expérience de minage en France.

Les modes de minage proposés
Solo-mining pur
Pour qui

Pour miner seul, sans devoir maintenir un node. Si vous avez déjà votre propre node, ce mode ne vous apportera pas davantage.

Conseil souveraineté : maintenir un node local est rapide, peu coûteux et apporte de vrais avantages. Nous le recommandons.
Connexion

stratum+tcp://chauffagistes-pool.fr:3333
AxeOS : URL : chauffagistes-pool.fr – Port : 3333
User : [Votre adresse de réception Onchain] — Password : x

Gains

La récompense du bloc tombe directement sur votre wallet, sans frais ni transit chez nous.

Avantages
  • 100% de la reward
  • Pas besoin de maintenir un node
Minage privé collaboratif
Pour qui

Pour miner entre amis / communauté, sans exposer un node public, et partager les gains équitablement.

Connexion

stratum+tcp://chauffagistes-pool.fr:3333
User : [Votre adresse Onchain].[identifiant] — Password : x
Exemple : bc1qkt...ae.Satoshi
(Ces identifiants sont publics si un bloc est trouvé.)

Gains & répartition (optionnelle)

La récompense tombe sur le wallet indiqué en User.
Répartition proposée : [Somme diff. de vos shares] / [Somme diff. du groupe] → pourcentage de participation effective sur la période.

Avantages
  • 100% de la reward, à partager comme vous voulez
  • Aucune gestion de node
  • Calcul clair des parts de travail
Minage en Pool
Pour qui

Pour maximiser les chances de trouver un bloc en mutualisant la puissance.

Connexion

stratum+tcp://chauffagistes-pool.fr:3333
User : bc1qqp9zq4an6nyzhcspz2xfmkcf8rj0p6w94a5gyeu2a7rghxjhnqqsvymz5m.[identifiant] — Password : x
(Identifiants visibles si un bloc est trouvé.)

Gains & sécurité

Les récompenses vont sur un wallet multi-signature détenu par plusieurs membres. Personne ne peut agir seul. Répartition calculée en temps réel :
[Somme diff. des shares d’un utilisateur] / [Somme diff. totale du groupe]

Avantages
  • Chances bien plus élevées de trouver un bloc
  • Répartition juste basée sur le travail effectif (PoW)
  • Évite de décourager les petits mineurs
À propos

Né d’un groupe qui découvre Bitcoin plus sérieusement depuis 2024, avec l’idée de se chauffer en minant et de garder une transparence maximale sur la répartition des récompenses. Nous utilisons la base open-source ckpool et avons ouvert nos travaux à la communauté depuis l’été 2025.

Aucun frais prélevé : nous vivons des dons et de nos propres mineurs dans la pool. Nous pensons que des petits dispositifs de minage domestiques auront leur place dans chaque foyer — d’où la nécessité d’outils simples, justes et transparents.

Partager, construire, apprendre.
